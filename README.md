Mad Libs Generator 🎭

A fun and interactive Mad Libs Generator built using Python. The program asks users for different types of words and inserts them into a story, creating a unique and often humorous result.

## Features
- Interactive user input
- Dynamic story generation
- Easy-to-customize story templates
- Beginner-friendly Python project
- Fun and engaging gameplay

## Technologies Used
- Python
- Variables
- String Formatting (f-strings)
- User Input Handling

## Learning Outcomes
- Working with user input
- Understanding variables and strings
- Using string interpolation with f-strings
- Building interactive Python applications

## How to Run
1. Open the project in Python.
2. Run the program.
3. Enter the requested words (noun, verb, adjective, etc.).
4. Read the generated story.

## Sample Gameplay

Enter a name: Rahul

Enter an adjective: funny

Enter a place: park

Generated Story:

One day, Rahul went to the park. It was a funny day full of surprises and adventures!
